# Cost_of_Drought_ABM
ABM model for master thesis, modeling drought risk to the foundation of houses 


In this github there are a few files:
 - The Model
 - model.py contains the actual model
   
 - The Agents
 - agents.py creates the agents and their actions within a step
   
 - The Agent attributes
 - agent_attributes.py creates the agent attributes and the synthetic population for the ABM
   
 - Run
 - run.py runs the model and saves it in a dataframe
   
 - Run for groups
 - run_groups.py does the same thing, but it runs on a group level
